# PANDUAN MENGGUNAKAN MENDELEY UNTUK JURNAL JPIT MARSHA

Halo Marsha! Master Irsyad telah menyiapkan semua bahan draf jurnal JPIT kamu dengan sangat lengkap dan rapi. Di dalam folder ini, terdapat file bernama `marsha_references.bib`.

File tersebut berisi 20 daftar pustaka jurnal internasional (IEEE) yang sudah kami kumpulkan otomatis untukmu. Kamu TIDAK PERLU mengetik nama pengarang atau judul jurnal satu per satu. 

Karena kamu menggunakan **Mendeley Desktop** atau **Mendeley Reference Manager**, ikuti panduan super mudah ini:

## LANGKAH 1: Memasukkan File .bib ke Mendeley
1. Buka aplikasi **Mendeley** di laptop kamu.
2. Klik tombol **Add New** (atau menu **File** -> **Import** di versi Mendeley lama).
3. Pilih opsi **Import library** lalu pilih **BibTeX (*.bib)**.
4. Cari dan pilih file `marsha_references.bib` yang ada di folder ini.
5. Selesai! Sebanyak 20 referensi ilmiah akan langsung masuk secara ajaib ke daftar pustaka Mendeley kamu lengkap dengan nama penulis, tahun, dan volumenya.

## LANGKAH 2: Mensitasi di Microsoft Word
Saat kamu sedang menempelkan draf "Bab Pendahuluan" atau "Bab Metode" dari file Markdown kami ke dalam Microsoft Word, kamu akan menemukan angka-angka seperti `[11], [12]` di akhir kalimat.

JPIT mewajibkan sitasi menggunakan Mendeley. Begini cara menggantinya:
1. Hapus ketikan manual `[11], [12]` di Word tersebut.
2. Letakkan kursor di akhir kalimat tersebut.
3. Buka tab **References** di Word, lalu klik **Mendeley Cite** (atau Insert Citation).
4. Di kolom pencarian Mendeley, ketik nama penulis atau potongan judul. (Peta/Cheat Sheet nama penulis yang harus diketik sudah kami sediakan di bagian paling bawah draf **marsha_04_Bab_Pendahuluan.md**).
5. Klik jurnal yang muncul, lalu tekan **Insert Citation**.
6. Angka `[1]` akan muncul secara otomatis dengan gaya IEEE!

## LANGKAH 3: Memuntahkan Daftar Pustaka JPIT
Setelah kamu selesai memasukkan semua sitasi dari paragraf awal sampai akhir:
1. Pindah ke halaman paling bawah pada file Word JPIT kamu, tepat di bawah judul bab **DAFTAR PUSTAKA**.
2. Di menu Mendeley Cite pada Microsoft Word, cari pilihan **Insert Bibliography** (tekan titik tiga `...` di Mendeley Cite panel kanan, lalu pilih *Insert Bibliography*).
3. Mendeley akan otomatis mencetak seluruh ke-20 daftar pustaka!
4. **PERHATIAN KHUSUS JPIT:** Blok seluruh teks daftar pustaka yang baru saja dicetak Mendeley, dan paksa ubah ukuran font-nya menjadi **Times New Roman ukuran 8 pt**. (Karena template JPIT mewajibkan font ukuran 8 untuk Daftar Pustaka).
