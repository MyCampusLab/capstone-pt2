## 🚀 BAGIAN 2: LANGKAH-LANGKAH EKSEKUSI (STEP-BY-STEP)

Silakan buka dokumen **Template Word JPIT 2024** di laptop kamu, lalu lakukan urutan ini tanpa melompat:

### Langkah 1: Persenjatai Mendeley Kamu
Buka file `PANDUAN_MENDELEY_MARSHA.md`. Ikuti instruksinya untuk memasukkan file `.bib` ke Mendeley Desktop kamu. Jangan lanjut ke Langkah 2 sebelum 20 daftar pustaka tersebut berhasil masuk ke Mendeley.

### Langkah 2: Mengisi Kepala Jurnal (Header)
Buka file `marsha_04_B_Header_dan_Abstrak.md`. 
* *Copy-paste* Judulnya ke Word. 
* Ganti tulisan [Nama Dosen Pembimbing] dengan dosenmu. 
* *Copy-paste* abstrak bahasa Inggris (Pastikan cetak miring / *Italic*).
* *Copy-paste* abstrak bahasa Indonesia (Pastikan teks normal).

### Langkah 3: Menyusun Bab Pendahuluan & Memasang Sitasi
Buka file `marsha_04_Bab_Pendahuluan.md`.
* *Copy-paste* teksnya (tanpa panduan bawahnya) ke Word.
* Di teks itu, kamu akan melihat angka seperti `[11], [12]`. Hapus teks itu, lalu gunakan plugin **Mendeley Cite** di Word untuk menggantikannya. 
* *Bingung harus cari nama jurnal apa di Mendeley?* Tenang, di bagian paling bawah file markdown tersebut, Irsyad sudah menyiapkan "Contekan Navigasi Mendeley". Kamu tinggal intip contekan itu untuk tahu jurnal mana yang harus diklik.

### Langkah 4: Menyusun Bab Metode & Rumus Matematika
Buka file `marsha_05_Bab_Metode.md`.
* *Copy-paste* teksnya ke Word.
* Pastikan sub-babnya (A, B, C) kamu jadikan cetak miring (*Italic*).
* Terdapat **Persamaan 1** dan **Persamaan 2**. Jangan biarkan berupa teks biasa! Ketik ulang rumus tersebut dengan cantik menggunakan menu `Insert -> Equation` di Word agar terlihat seperti tesis betulan.
* **TUGAS KAMU:** Di tempat bertuliskan `(Tempatkan Gambar 1 Di Sini)`, buatlah satu diagram kotak/panah (*flowchart*) sederhana di Canva/Visio yang menggambarkan alur aplikasinya (Kamera -> AI -> Layar Blur). Lalu letakkan gambarnya di tengah halaman Word.

### Langkah 5: Menyusun Bab Hasil & Tabel Pembuktian
Buka file `marsha_06_Bab_Hasil.md`.
* *Copy-paste* teks dan tabelnya.
* **SANGAT PENTING:** Edit tabel tersebut di Word. Aturan JPIT melarang keras ada garis tegak (vertikal) di dalam tabel. Sisakan garis horizontal (mendatar) saja di bagian atas dan bawah.
* **TUGAS KAMU:** Di tempat bertuliskan `(Tempatkan Gambar 2 Di Sini)`, masukkan satu foto/screenshot aplikasi VisionSafe saat layar sedang menjadi *Blur* (sebagai bukti intervensi visual).

### Langkah 6: Menyusun Penutup
Buka file `marsha_07_Bab_Simpulan.md`.
* *Copy-paste* Bab Simpulan (Ini cuma 1 paragraf panjang, jangan pecah jadi poin-poin).
* Pergi ke halaman paling akhir di Word (di bawah judul DAFTAR PUSTAKA).
* Gunakan Mendeley Cite (`Insert Bibliography`) untuk memuntahkan semua daftar pustaka yang sudah kamu sitasi di Langkah 3 secara otomatis.
* **SANGAT PENTING:** Blok teks daftar pustaka yang keluar dari Mendeley itu, dan kecilkan ukuran font-nya menjadi **8 pt**! (Bawaan Word biasanya 10pt atau 11pt, JPIT mewajibkan 8pt).
