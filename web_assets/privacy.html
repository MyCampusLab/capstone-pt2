VisionSafe
Kebijakan Privasi & Perlindungan Data (COPPA Compliant)

1. Perlindungan Anak Sepenuhnya

VisionSafe dirancang dari awal untuk menjadi lingkungan yang sangat aman bagi anak. Kami sepenuhnya mematuhi Children's Online Privacy Protection Act (COPPA). Aplikasi ini tidak pernah mengunggah, menyimpan, atau mengirimkan gambar wajah anak ke server cloud manapun.

2. Penggunaan Kamera Lokal (Edge AI)

Izin kamera semata-mata digunakan untuk algoritma Machine Learning lokal (MediaPipe) pada perangkat (HP) Anda untuk mengukur jarak wajah dan mendeteksi postur. Setelah frame video diproses menjadi koordinat angka matematis secara real-time, frame tersebut langsung dimusnahkan di memori lokal.

3. Pengumpulan Data Telemetri

Data yang kami kirim ke Database Supabase hanyalah data agregasi anonim, seperti:

  • Durasi Sesi Layar (Menit)
  • Rata-rata Jarak Baca (CM)
  • Jumlah Teguran (Kali)
  • Pencapaian Gamifikasi (XP & Level)

4. Penghapusan Akun & Data

Sesuai standar Google Play, Anda memiliki kendali penuh atas data Anda. Anda dapat menggunakan fitur "Hapus Akun & Data" di menu Pengaturan aplikasi untuk memusnahkan seluruh riwayat telemetri, identitas, dan kredensial Anda dari server Cloud kami secara permanen.

5. Pemantauan Latar Belakang (Foreground Service)

Untuk melindungi mata anak secara berkelanjutan, VisionSafe menggunakan layanan latar depan (Foreground Service) yang terikat dengan notifikasi sistem. Layanan ini memungkinkan kami memantau jarak mata melalui kamera secara terus-menerus meskipun aplikasi disembunyikan atau pengguna sedang menggunakan aplikasi lain. Pemantauan background ini sangat krusial agar perlindungan tidak terputus. Sama seperti pemrosesan reguler, pemrosesan latar belakang ini tetap berjalan 100% secara lokal dan tidak ada satupun rekaman gambar yang disimpan atau dikirim ke internet.
